## Overview

Vastra AI is an innovative AI-powered virtual try-on application specifically designed for Indian ethnic wear. Using advanced machine learning and computer vision, it allows users to virtually try on traditional Indian garments like sarees, sherwanis, and other ethnic attire with photorealistic accuracy.

## Features

- **Virtual Try-On**: Experience photorealistic draping of ethnic garments on your own body
- **Body Analysis**: AI-powered biometric mapping for personalized recommendations
- **Smart Wardrobe**: Save and organize your favorite virtual looks
- **Ethnic Collections**: Curated catalog of premium Indian ethnic wear
- **Multi-Angle Views**: Front, side, back, close-up, and 360-degree views
- **Color Variants**: Explore different color options for garments
- **User Authentication**: Secure sign-up and login system
- **Cart & Checkout**: Purchase functionality for selected items

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **AI/ML**: Google Gemini AI (Gemini 3 Flash, Gemini 2.5 Flash Image)
- **Styling**: Tailwind CSS (via custom components)
- **State Management**: React hooks with localStorage persistence
- **Build Tool**: Vite

## How It Works

1. **User Registration**: Create an account and upload a professional portrait
2. **Body Analysis**: AI analyzes your body shape, skin tone, and features
3. **Garment Selection**: Browse curated ethnic collections
4. **Virtual Try-On**: AI generates photorealistic images of you wearing selected garments
5. **Customization**: Explore different angles and color variants
6. **Save & Purchase**: Save looks to your wardrobe or add to cart

## Prerequisites

- Node.js (v18 or higher)
- Google Gemini API key

## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd vastra-ai---indian-ethnic-try-on
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**

   Navigate to `http://localhost:5173` (default Vite port)

## Usage

1. **Register/Login**: Create an account or sign in
2. **Upload Photo**: Upload a clear, well-lit portrait for body analysis
3. **Explore Collections**: Browse through curated ethnic garments
4. **Try On**: Select garments to see virtual try-on results
5. **Customize**: Switch between different angles and colors
6. **Save Looks**: Save your favorite combinations to your wardrobe

## API Integration

The app integrates with Google Gemini AI for:
- Body shape and feature analysis
- Photorealistic image generation
- Virtual garment draping
